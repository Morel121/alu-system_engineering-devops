# System Engineering & DevOps - Web Infrastructure: Firewall

## Description
This repository contains configuration scripts and documentation for managing network security using **UFW (Uncomplicated Firewall)** on Ubuntu servers as part of the ALU DevOps curriculum.

The project focuses on configuring firewall rules to secure `web-01` by restricting unneeded network ports while maintaining necessary access for SSH, HTTP, and HTTPS traffic.
